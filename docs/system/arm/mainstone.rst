Intel Mainstone II board (``mainstone``)
========================================

The ``mainstone`` board emulates the Intel Mainstone II development
board, which uses a PXA270 CPU.

Emulated devices:

- Flash memory
- Keypad
- MMC controller
- 91C111 ethernet
- PIC
- Timer
- DMA
- GPIO
- FIR
- Serial
- LCD controller
- SSP
- USB controller
- RTC
- PCMCIA
- I2C
- I2S
